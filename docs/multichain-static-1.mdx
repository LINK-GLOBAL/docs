## Generating Multichain Static Accounts

To retrieve a Multichain Static Account, call the **multichain\_static** endpoint.

You'll need to:

* Designed for **businesses or enterprises** serving businesses (B2B).
* Supports deposits in multiple currencies with **local banking details** (e.g., account numbers, routing codes).
* Supports **payouts to multiple blockchain networks.**

LINK automatically handles the fiat-to-stablecoin conversion and sends the funds on-chain to the designated wallet.

---
title: "Setup"
description: "To set up webhooks URL on your Bridge account, follow the steps below:"
---

1. Ensure you already have access to Bridge. If you don’t, head over to [Bridge](https://app.linkio.world/signin) and create one.

2. **Navigate to the developer page** on the side bar panel

3. **Add Webhooks:** Click the “Add Webhook” button to create a new webhook

![](/images/docs/6a846aee4acb45bb1576a26a9defd10b409038f2285fa357ab0fe53c1a664d0c-Screenshot_2025-10-13_at_15.40.27.png)

4. **Webhook details:** input your webhook URL in the input field to receive live webhook events.

![](/images/docs/897d0f9cc4fed0cb4c4db93ec92b7f20ae6c2154274fccc5e8b872b3e3c47fb4-Screenshot_2025-10-13_at_15.41.18.png)

<Info>
  ### NOTE: Your URL must end with "/webhook".
</Info>

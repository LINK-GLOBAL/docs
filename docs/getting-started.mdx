### OVERVIEW

LINK is a financial technology company building FX and stablecoin payment infrastructure for global businesses.

With LINK, you can:

* On-ramp and off-ramp stablecoins (i.e., USDC, USDT) via local bank transfers and global payment methods

* Settle payouts in multiple markets using stablecoin wallets and bank accounts through a single, unified API

***

### Quickstart guides

* **Create Static Payment Templates** – Easily create onramps and offramps using our orchestration API

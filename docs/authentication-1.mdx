### Secret Key

Your **Secret Key** is used to authenticate all API requests. It can be found in your LINK Bridge Dashboard under the **Developers** section.

<Info>
  ### **NOTE:** All API requests require **Bearer Token authentication** using your Secret Key.
</Info>

![](/images/docs/02b9fb0427385021ff0a67b5d7a08a2d4a97c213877c3365491a1bab494d9266-Screenshot_2025-08-19_at_21.22.49.png)

***

### Get set up with LINK Bridge

**Step 1: Create a LINK Bridge account** Go to [https://app.linkio.world](https://app.linkio.world) and create your free developer account. You’ll use the LINK Bridge Dashboard to manage payments and access your API settings through an intuitive interface.

***

**Step 2: Create your API keys** Once logged in, navigate to the **Developers** tab in the left menu and click **Generate New Key.**

<Info>
  ### **NOTE:** You can regenerate your keys at any time from your Dashboard.
</Info>

<Warning>
  ### **IMPORTANT:** Never share your Secret Key publicly or expose it in client-side code.
</Warning>

***

**Step 3: Follow our guides and start moving money.**

## Banking Object Requirements

When creating a deposit request, use these banking details to make the transfer depending on the **currency** and **payment route** (domestic vs. international).

Use the table below to find the exact fields required for each object. These fields must be provided as strings in your JSON payload.

| Object Name            | Currency | Route / Region           | Required String Fields                                                                                          |
| ---------------------- | -------- | ------------------------ | --------------------------------------------------------------------------------------------------------------- |
| usd\_domestic          | USD      | Domestic                 | account\_number, routing\_number, ach\_routing\_number, fedwire\_routing\_number, bank\_name, beneficiary\_name |
| usd\_international\_us | USD      | International (US)       | account\_number, ach\_routing\_number, fedwire\_routing\_number, swift\_code, bank\_name, beneficiary\_name     |
| usd\_international\_eu | USD      | International (EU)       | iban, swift\_code, bank\_name, beneficiary\_name                                                                |
| usd\_international\_sg | USD      | International (SG)       | account\_number, swift\_code, bank\_code, branch\_code, bank\_name, beneficiary\_name                           |
| usd\_international\_hk | USD      | International (HK)       | account\_number, swift\_code, bank\_code, branch\_code, bank\_name, beneficiary\_name                           |
| eur\_domestic          | EUR      | Domestic                 | iban, bank\_name, beneficiary\_name                                                                             |
| eur\_international\_eu | EUR      | International (EU)       | iban, swift\_code, bank\_name, beneficiary\_name                                                                |
| eur\_international\_sg | EUR      | International (SG)       | account\_number, swift\_code, bank\_code, branch\_code, bank\_name, beneficiary\_name                           |
| gbp\_domestic          | GBP      | Domestic (UK)            | account\_number, sort\_code, iban, bank\_name, beneficiary\_name                                                |
